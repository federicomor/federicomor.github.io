Problem 3

The prime factors of $13195$ are $5, 7, 13$ and $29$.

What is the largest prime factor of the number $600851475143$?



Soluzione

Qui propongo tre metodi: il primo one-liner sfruttando le potenzialit√† offerte da Julia (moralmente, la libreria Primes); il secondo usando il primo metodo che (credo) verrebbe in mente, ma risulta molto costoso, inefficiente; il terzo una versione sempre a mano ma pi√π ottimizzato e ragionato.

# metodo one-liner (usando le potenzialit√† di Julia)
using Primes
maximum(keys(factor(n)))

# metodo naive (il primo che verrebbe in mente)
function find_max_pfactor_naive(n)
	lim_initial = div(n,2)
	for i in lim_initial:-1:1
		if isprime(i) && n%i == 0
			@show i
			break
		end
	end
end
find_max_pfactor_naive(n)

# metodo efficiente
function find_max_pfactor_fast(n)
	i = 1
	while n != 1
		i += 1
		if n % i == 0
			println("n=$(rpad(n,20)) -> $i √® un fattore primo di n")
			while n % i == 0
				n = n/i
			end
		end
	end
	println("n=$(rpad(n,20)) -> finito!")
	println("$i √® il pi√π alto fattore primo di n")
end
find_max_pfactor_fast(n)
